# Obrázky pro web Areál Zastávka

## Logo

Logo webu by mělo být uloženo jako:
- **logo.png** - hlavní logo s transparentním pozadím
- Umístění: `public/images/logo.png`
- Doporučené rozměry: minimálně 400x96px (poměr stran cca 4:1)
- Formát: PNG s transparentním pozadím

Logo obsahuje:
- Průmyslové budovy a ozubená kola na levé straně
- Text "AREÁL ZASTÁVKA" na pravé straně
- Barvy: tmavě modrá, tmavě šedá, červeno-hnědá

## Obrázky areálu

Obrázky budov areálu jsou v složce `areal/`:
- `areal-zastavka.jpg` - letecký pohled
- `hala-6-5.jpg` - budova HALA 6.5
- `budova-terakota.jpg` - budova s terakotovou fasádou
- `budova-hneda.jpg` - budova se světle hnědou fasádou
